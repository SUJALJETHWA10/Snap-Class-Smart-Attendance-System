# 🎓 Snap-Class Smart Attendance System

Snap-Class Smart Attendance System is an **AI-powered attendance management platform** that automates student attendance using **Face Recognition** and **Voice Recognition** technologies. The system provides a smart, secure, and efficient way to mark attendance, reducing manual work and improving classroom management.

Built using **Streamlit**, **Machine Learning**, **Computer Vision**, and **Voice Biometrics**, the platform enables real-time student authentication, attendance tracking, QR generation, and secure data management.

---

## 🚀 Features

### 👤 Face Recognition Attendance
- Real-time student face detection and recognition
- Automatic attendance marking using facial authentication
- Fast and secure identity verification
- Reduced proxy attendance issues

### 🎤 Voice Recognition Attendance
- Voice-based student authentication
- Speaker recognition using voice biometrics
- Alternative attendance option for accessibility

### 📊 Attendance Management
- Automated attendance records
- Student attendance tracking
- Efficient classroom management
- Real-time attendance updates

### 🔐 Secure Authentication
- Password hashing using **bcrypt**
- Secure user authentication
- Student profile management

### 🔳 QR Code Generation
- Dynamic QR code generation using **Segno**
- Quick attendance access and verification

### 📈 Data Handling
- Efficient student data management
- Data analysis and processing using **Pandas** and **NumPy**
- Cloud database integration with **Supabase**

## 📂 Project Workflow

1. **Student Registration**
   - Register student details
   - Capture face data
   - Store voice samples

2. **Authentication Process**
   - Face recognition verifies identity
   - Voice recognition validates student authentication

3. **Attendance Marking**
   - Attendance is automatically recorded
   - Student records are updated in real time

4. **Attendance Management**
   - Teachers can monitor attendance
   - View attendance reports and logs
